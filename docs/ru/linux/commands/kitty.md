# Kitty #

**Kitty** is a GPU-accelerated terminal emulator for UNIX-like systems, known for its speed, low latency, and modern features, for example, image and Unicode rendering, tabs, windows.

## Management ##

### Tab Management ###

- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>T</kbd> — Open a new tab.
- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Q</kbd> — Close the current tab.
- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>→</kbd> / <kbd>←</kbd> — Switch to the next / to the previous tab.
- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>.</kbd> — Move a tab forward.
- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>,</kbd> — Move a tab backward.

### Window Management ###

- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Enter</kbd> — Split a window horizontal.
- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>W</kbd> — Close the current window.
- <kbd>Ctrl</kbd> + <kbd>Tab</kbd> — Switch to the next window.
- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Tab</kbd> — Switch to the previous window.

### Font Adjustments ###

- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>+</kbd> — Increase a font size.
- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>-</kbd> — Decrease a font size.
- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Backspace</kbd> — Reset a font size to default.

### Clipboard ###

- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>C</kbd> — Copy selected text.
- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>V</kbd> — Paste from clipboard.

### Miscellaneous ###

- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>LMC</kbd> — Open the URL under cursor.
- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>F11</kbd> — Toggle fullscreen.
- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>F5</kbd> — Reload the configuration file.
